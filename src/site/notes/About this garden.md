The notes on this website are written using [Obsidian](https://obsidian.md/), a tool for taking notes in [markdown](https://en.wikipedia.org/wiki/Markdown), and are published to the web using [Obsidian Digital Garden](https://github.com/oleeskild/obsidian-digital-garden) and [Vercel](https://vercel.app/).

The appearance of this website (the "theme") is called [Primary](https://github.com/primary-theme/obsidian).
