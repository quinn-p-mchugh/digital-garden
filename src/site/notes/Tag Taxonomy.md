This page documents the various [tags](https://help.obsidian.md/tags) used on this site and what they mean.

# Navigation-Related
- #index - a note used to help structuee and navigate through notes about a certain topic, like the `Index` of a textbook. This is kind of note is commonly referred to as a [map of content](https://obsidian.rocks/maps-of-content-effortless-organization-for-notes/) in the Obsidian community.

# Task-Related
- #to/ - a general tag to indicato there is somethijgn
	- #to/do - used in to indicate there is something to be done in this note/section e.g. `#to/do Resolve formatting issues`
	- #to/flesh-out - used to indicate that the content of a note or section needs to be fleshed out in some way
