This page documents the various [tags](https://help.obsidian.md/tags) used on this site and what they mean.

- #index - a note used to help structure and navigate through notes about a certain topic. Similar to an index in a book.

- #to/
	- #to/flesh-out - used to indicate that a section or note needs to be fleshed out
